# KHELLO KARACHI ⚡️
*Karachi's first unified platform to reserve sports grounds and challenge local teams.*

## 📖 About The Project

Organizing a grassroots sports match in Karachi is currently broken—requiring endless phone calls to turf managers, tracking down players who drop out, and dealing with double-booked venues. **Khello Karachi** solves this by centralizing facility management and peer-to-peer matchmaking into a single, high-performance web application. 

Built specifically for the local market, Khello connects players, team captains, and facility managers for Futsal, Padel, and Cricket.

### ✨ Core Features
* **Smart Court Booking:** Browse real-time availability, secure time slots, and manage advance payments seamlessly.
* **Peer-to-Peer Matchmaking:** Host matches, or challenge rival squads. 
* **Accountability & Refund Engine:** A robust, state-machine-driven dispute and refund workflow that bridges external banking transfers with in-app verifications.
* **Dual Dashboards:** Specialized, isolated environments for Players (to track history and teams) and Court Managers (to process bookings, manage revenue, and handle refunds).

### 🛠 Tech Stack
* **Frontend:** React.js (Vite), Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Mongoose)
* **Media & Storage:** Cloudinary, Multer
